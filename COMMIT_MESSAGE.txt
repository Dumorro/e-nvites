feat: implementa sistema de envio de email automático pós-confirmação

Adiciona sistema completo de envio de email de confirmação após o convidado
confirmar presença, replicando o mecanismo do projeto qr-gen.

## Funcionalidades Implementadas

### Envio Automático
- Email enviado automaticamente após confirmação via /api/rsvp/confirm-by-email
- Processo não-bloqueante (confirmação sempre sucede mesmo se email falhar)
- Sistema de retry automático (1 tentativa adicional)
- Logs detalhados de envio no console

### Template de Email
- Template React bilíngue (Português/Inglês)
- Design responsivo com cores da marca Equinor (#FF1243)
- QR Code gerado dinamicamente e incluído inline (base64)
- Informações completas do evento (nome, data, hora, local)
- Link personalizado para página de confirmação

### Painel Admin
- Botão "Reenviar Email" na lista de convidados
- Disponível apenas para convidados confirmados com email
- Feedback visual durante envio
- Confirmação antes de reenviar

### Sistema de Logging
- Tabela email_logs no Supabase
- Registra status (sent/failed/pending)
- Mensagens de erro detalhadas
- Timestamp de envio

## Configuração Necessária

Antes de usar em produção:
1. Criar conta no Resend (resend.com)
2. Obter API key
3. Configurar variáveis de ambiente:
   - RESEND_API_KEY
   - SMTP_FROM_EMAIL
   - SMTP_FROM_NAME
   - NEXT_PUBLIC_SITE_URL
4. Executar migration SQL (tabela email_logs)
